    npm run dev:simple

3️⃣  Dans UN AUTRE TERMINAL, démarrer le frontend :

    cd /mnt/c/Users/abdou/Desktop/sengp
    npx http-server www -p 8100

│  🔧 SI PROBLÈME                                             │
└─────────────────────────────────────────────────────────────┘

PostgreSQL ne démarre pas ?
    sudo service postgresql start

Redis ne démarre pas ?
    redis-server
    (ou ignorez, c'est optionnel)

Le backend ne démarre pas ?
    1. Vérifiez que PostgreSQL fonctionne : pg_isready
    2. Exécutez ./SETUP_DATABASE.sh
    3. Vérifiez les logs : cat backend/logs/combined.log

Erreur "Table users does not exist" ?
    ./SETUP_DATABASE.sh
